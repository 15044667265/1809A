## 实训练习笔记

### [git操作](https://jasonandjay.github.io/study/zh/standard/Start.html#git%E7%8E%AF%E5%A2%83)
- 创建仓库
- 查看状态
- 提交信息
  - 初始化：init
  - 新增：add
  - 更新：update
  - 修复：fix
  - 删除：remove
  - 完成：finish
- 撤销
  - 软撤销 git reset --soft [commit:6]
  - 硬撤销 git reset --hard [commit:6]
- 分支
  - 创建分支
  - 切换分支
  - 删除分支
  - 合并分支
- 提交
